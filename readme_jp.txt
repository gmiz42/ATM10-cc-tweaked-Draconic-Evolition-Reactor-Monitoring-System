１．概要
本プログラムは、Minecraft Modpack ATM10 ver6.2.1におけるDraconic Evolutionの
Reactor（Draconic Reactor）をcc:tweakedを使って監視するシステムです。

２．動作確認環境
Minecraft Modpack: All The Mod 10 ATM10 ver6.2.1
追加使用Mod：Draconic Evolution Render Patcher 1.1.2
https://www.curseforge.com/minecraft/mc-mods/draconic-evolution-render-patcher
そのままではリアクターが表示されません（表示されないだけで動きはする）ので、
表示したい方はこのModを導入してください。


３．インストール
方法は3つあります。
・ダウンロードしたluaファイルを、computercraftフォルダ内にあるコンピューターのIDのフォルダにluaを直接コピーしてください。
（例：AlltheMods10ATM10\saves\ワールド名\computercraft\computer\コンピューターのID）
　コンピューターのIDのフォルダは、editコマンドで何か適当にファイルを保存すると作成されます。
・特にインターネット接続に制限がない場合は、wgetコマンドを使ってgithubからダウンロードしてください。
　外部（モニタリング用）PC
　wget https://raw.githubusercontent.com/gmiz42/ATM10-cc-tweaked-Draconic-Evolition-Reactor-Monitoring-System/main/install_outside.lua install.lua
install
　内部（情報収集・送信用）PC
　wget https://raw.githubusercontent.com/gmiz42/ATM10-cc-tweaked-Draconic-Evolition-Reactor-Monitoring-System/main/install_inside.lua install.lua
install
　ダウンロード後、各PCの画面から install を実行。

・ダウンロードしたファイルを現実のPCにあるcc:tweakedディスクドライブ用フォルダにコピーし、（例：.minecraft/saves/.../disk/ID/）
　ディスクドライブを使い、コンピューターのfsコマンドでコピーしてください。
参考：https://tweaked.cc/module/fs.html

４．セットアップ
前提条件として、Draconic Evolutionのカオスシャード（カオスの破片）、つまりカオスガーディアンの撃破が必要です。

	・CompactMachine SoarynまたはFarming
	　リアクターは、拠点とは違うディメンジョンに作成しましょう。

	・必要なローディングチャンク：1個分
	　1チャンク以内に収まるようにセットアップすることをお勧めします。

	・DraconicEvolutionで必要なブロック
	　リアクター本体 : 1
	　リアクター安定化装置 : 4
	　リアクターインジェクション : 1
	　フラックスゲート : 2

	・出力先
	　高性能な蓄電装置
	　例）Draconic Evoliton エナジーコア、Mekanismインダクションマトリックス、Applied Flux（AE2アドオン）のフラックスアクセッサ―（256M エネルギーセル）など

	・cc:tweakedで必要なブロック
	　・高度なコンピューター : 2
	　　リアクター安定化装置に接続するPCとモニターに接続するPCの2台運用を推奨。
	　・エンダーモデム : 2
	　　PCにくっつけて使用します。PC間通信、特にディメンジョンをまたいだ通信ができます。
	　・有線モデム（ブロック） : 1個以上
	　　Draconic Evolutionの機械に対して設置します。ハブ的な使い方ができるので、このブロックまたはPCに接続している機械に対してはモデム不要です。
	　・有線モデム : 2個以上
	　　おもにモニターとの通信ですが、リアクターインジェクションに接続したフラックスゲートとの通信に使用します。
	　・cc:tweakedのネットワークケーブル : 必要なだけ
	　・高度なモニター : 高さ 4個分 x 幅 8個分
	　PC、入力・出力担当のフラックスゲート、モニターそれぞれにモデムを設置してお互いをケーブルでつなぎ、すべてのモデムを右クリックして有効化してください。
	　

	・保険
	　(Mekanism)ダンボール
	　小爆発が起こったら、リアクター中心部をダンボールで右クリックすると梱包できます。


５．使い方（運用方法）
　１．コンピューターの画面を開き、peripheralsコマンドで接続機器の名前を確認してください。
　２．コンフィグファイルを開き、対応する機器の名前を入力してください。
　３．プログラムファイルの名前をstartup.luaに変更してください。
　　　これでゲームを再起動した際に自動で起動します。　
　４．モニタのモードボタンを右クリックで触るとモードを切り替えることができます。
　
★画面のステータスについて
左側の棒グラフはそれぞれ左から現在の温度、現在のフィールド強度、現在の飽和度を示しています。
右側の折れ線グラフは温度、飽和度、フィールド強度、予測残り稼働時間、燃焼レートの推移を表示しています。
画面下部の情報は
上段　推奨設定要求出力、最大要求出力までの余裕、現在のモード（右クリックで切り替え可能）
中段　稼働ステータス、設定出力、投入燃料と反応物の割合（及び残り燃料割合）
下段　現在の温度、燃焼レート（ナノバケツ換算）、残り稼働時間
となっています。

Draconic Reactorは
・温度が高いと爆発
・入力エネルギーが低いと爆発
・電力を要求しすぎると爆発
・ほったらかすと爆発
・爆発させると広範囲が消失する
という危険極まりない発電機構です。

Draconic Reactorを安全に運用しようというコンセプト、そして
リアクターってなんかカッコイイよな！という浪漫のもとに以下の機能を実装しています。
・グラフィカルなUI
・推奨出力の提案
・残り時間推定
・限界出力の推定
・モード切替による安全運用/攻めた運用の切り替え

また、このプログラムを使って監視しても爆発するときは爆発します。
安全で高出力で自動化可能な発電機構はほかにたくさんあります。
ロマンを求める方はご活用ください。
